# 專案名稱
React Date Picker

### 取得專案

```bash
git clone https://github.com/sihyun-user/React-Date-Picker.git
```

## 安裝

以下將會引導你如何安裝此專案到你的電腦上。

Node.js 版本建議為：`18.16.0` 以上

## 安裝套件

```bash
npm install
```

### 運行專案

```bash
npm run dev
```

### 開啟專案

在瀏覽器網址列輸入以下即可看到畫面

```bash
http://localhost:5173/
```
